Vibella - AI-Powered Instagram Assistant 
An elegant AI chatbot that helps Instagram users generate captions, hashtags, and song suggestions with a soft, aesthetic personality.

Project Overview:
Vibella is a semester project built with FastAPI, LangChain, and modern web technologies. It provides Instagram content creators with AI-powered assistance in a beautiful, minimal interface.

Key Features:
AI-generated Instagram captions
Relevant, non-spammy hashtags
Song suggestions for Instagram stories
Conversation history stored in Mongodb
Soft, aesthetic, modern UI design
Responsive design for all devices

Technology Stack
Backend:
FastAPI - Modern Python web framework
LangChain - LLM orchestration and prompt management
Mongodb - Lightweight database for conversation storage
Google Gemini 2.5 Flash - Fast, free, and powerful AI model

Frontend:
HTML5 - Structure
CSS3 - Styling with gradients and animations
JavaScript - Dynamic interactions
Google Fonts - Playfair Display & Poppins
